# Introduction
    Activations as Features (AAF) is an exploratory project dedicated to using intermediate layer activations from large language models (LLMs) as input features in cross-prompt automatic essay scoring tasks. The project is implemented in a Python 3.9 environment.

# File Functions
    att_map_and_context.py  ------  Drawing word score heatmap.
    config.py               ------  Parameters.
    head_act_reg.py         ------  Fit probes and plot scatter plots and heatmaps.
    LLM_activations.py      ------  Obtain LLM activation.
    probe.py                ------  Nonlinear probe class.
    qwk_cal.py              ------  QWK score calculation function.
    trait_direction.py      ------  Calculate directions under different prompts or traits.
    utils.py                ------  Utility function.
